# [Nom de l'Agence] — Template Site Vitrine Agence Créative Premium

## 🎯 Client cible
Agence de communication, studio de design, designer indépendant, consultant créatif ou agence de branding souhaitant une présence web sophistiquée et différenciante. Style luxe, minimaliste et élégant.

## ⚡ Personnalisation rapide (30 min)
1. **Couleurs** → Modifier `--color-primary` (noir), `--color-accent` (champagne `#C9A96E`), `--color-bg` (beige `#F5F0E8`) dans `:root` de `style.css`
2. **Textes** → Remplacer tous les `[Nom de l'Agence]`, `[Ville]`, `[Année]` dans `index.html` et `prestations.html`
3. **Logo** → Modifier `.logo` dans les deux fichiers HTML (`[Nom.]` + la couleur du point)
4. **Images** → Remplacer les URLs Unsplash dans `style.css` (`.section-hero background` + `.about-image background`)
5. **Services** → Modifier les `.card` dans `index.html` et `prestations.html`
6. **Tarifs** → Modifier les `.pricing-card` dans `prestations.html`
7. **Réseaux sociaux** → Remplacer les `href="#"` des `.social-links` (Instagram, LinkedIn, Dribbble, Behance)
