# ShootEvent Template — Site Vitrine Photographe / Vidéaste Événementiel

## 🎯 Client cible
Photographe événementiel, vidéaste de mariage, agence photo/vidéo ou prestataire d'image souhaitant une présence web premium avec galerie, tarifs et formulaire de contact.

## ⚡ Personnalisation rapide (30 min)
1. **Couleurs** → Modifier les variables CSS dans `style.css` ou `config.js`
2. **Textes & Nom** → Modifier directement dans `index.html` (nom, ville, slogan)
3. **Photos de portfolio** → Remplacer les images dans `/assets/` par les vrais clichés du photographe
4. **Formules & tarifs** → Modifier la section pricing dans le HTML
5. **Formulaire de contact** → Configurer l'endpoint dans `config.js` ou le HTML
6. **Réseaux sociaux** → Mettre à jour les liens Instagram, Facebook, YouTube

## 💰 Estimation commerciale
- **Fourchette de prix conseillée** : 1 500€ – 4 000€ selon les options
- **Temps de personnalisation estimé** : 4 à 8 heures (selon volume galerie)
- **Options vendables en upsell** :
  - Galerie lightbox avancée avec tri par album (+400€)
  - Intégration calendrier de réservation (+300€)
  - Blog / coulisses des shootings (+500€)
  - Témoignages clients avec photos (+250€)
  - Animation particle.js / cursor custom (+300€)
  - Page Before/After (avant/après retouche) (+400€)

## 🚀 Déploiement
1. Personnaliser `config.js` pour les coordonnées, textes et API
2. Remplacer toutes les images dans `/assets/` par les tirages réels
3. Configurer le formulaire de contact (voir `QUICKSTART.md`)
4. Héberger sur Netlify ou hébergement PHP mutualisé
5. Activer HTTPS et configurer le domaine
