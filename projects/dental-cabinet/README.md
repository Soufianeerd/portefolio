# [Nom du Cabinet] — Template Site Vitrine Cabinet Dentaire

## 🎯 Client cible
Chirurgien-dentiste, orthodontiste ou cabinet dentaire de groupe souhaitant une vitrine web professionnelle et rassurante. Idéal pour un cabinet moderne voulant recruter de nouveaux patients en ligne.

## ⚡ Personnalisation rapide (30 min)
1. **Couleurs** → Modifier `--color-primary` (bleu actuel `#4A90D9`) dans `:root` de `style.css`
2. **Textes** → Rechercher `[Nom du Cabinet]`, `[Ville]`, `[Adresse]` dans `index.html` et `prestations.html`
3. **Images** → Remplacer les URLs Unsplash dans `style.css` (`.section-hero` et `.about-image`)
4. **Services** → Modifier les `.card` dans `index.html` + `prestations.html`
5. **Tarifs** → Modifier les `.pricing-card` dans `prestations.html`
6. **Contact** → Remplacer l'endpoint Formspree dans `index.html` et configurer le numéro de téléphone
7. **Logo** → Remplacer l'icône `fa-tooth` et le texte `[Nom du Cabinet]` dans la nav et le footer
