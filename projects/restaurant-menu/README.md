# [Nom du Restaurant] — Template Menu Interactif Restaurant

## 🎯 Client cible
Restaurant gastronomique, brasserie haut de gamme, table d'hôtes ou bistro de chef souhaitant présenter sa carte en ligne de manière interactive et élégante.

## ⚡ Personnalisation rapide (20 min)
1. **Couleurs** → Modifier `--color-primary` (terre cuite `#8B4513`) et `--color-accent` (ocre `#D4A373`) dans `:root` de `style.css`
2. **Nom du restaurant** → Remplacer tous les `[Nom du Restaurant]`, `[Nom.]`, `[Ville]`, `[Adresse]` dans `index.html` et `prestations.html`
3. **Image hero** → Modifier l'URL Unsplash dans `.menu-header` dans `style.css`
4. **Plats et prix** → Modifier les `.menu-item` dans `index.html` (15 plats répartis en 4 catégories)
5. **Formules** → Modifier les `.pricing-card` dans `prestations.html`
6. **Catégories** → Ajouter/supprimer des `.cat-btn` et adapter les `data-category` correspondants
7. **Téléphone** → Remplacer le `href="tel:+33XXXXXXXXX"` dans les deux fichiers HTML
