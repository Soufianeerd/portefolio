# Restaurant La Rose — Template Site Vitrine Restaurant Gastronomique

## 🎯 Client cible
Restaurant traditionnel ou gastronomique français souhaitant une présence web élégante avec réservation en ligne. Idéal pour une brasserie haut de gamme, un restaurant de terroir ou un chef indépendant.

## ⚡ Personnalisation rapide (30 min)
1. **Couleurs** → Modifier `--color-primary`, `--color-accent`, `--color-bg` dans `:root` de `style.css`
2. **Textes** → Rechercher les `<h1>`, `<h2>`, `<p>` dans `index.html` et `prestations.html`
3. **Images** → Remplacer les URLs Unsplash dans `style.css` (`.section-hero background-image`)
4. **Menu & Prestations** → Modifier les `.menu-card` dans `index.html` + les `.card` dans `prestations.html`
5. **Formules tarifaires** → Modifier les `.pricing-card` dans `prestations.html`
6. **Contact** → Remplacer l'endpoint Formspree dans `index.html` (`action="https://formspree.io/f/XXXXXXXX"`)
7. **Logo** → Remplacer la balise `.logo` + le texte `Restaurant La Rose`
8. **Réseaux sociaux** → Remplacer les `href="#"` des `.social-links`

## 🚀 Déploiement
1. Personnaliser les fichiers selon le cahier des charges client
2. Configurer Formspree ou EmailJS pour le formulaire de réservation
3. Héberger sur Netlify (gratuit), OVH ou o2switch
4. Configurer le nom de domaine du client
5. Activer HTTPS (inclus Netlify / Let's Encrypt)
